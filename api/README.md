# E-Learning project (NodeJS REST API)

- Typescript
- MySQL

## Setup

Make sure to install the dependencies:

```bash
npm install
```

prisma create migrate

```
npx prisma migrate dev --name init
```

prisma pull migrate

```
npx prisma db pull
```

prisma push migrate

```
npx prisma db push
```

```
npm generate
```

## Development Server

Start the development server on `http://localhost:6020`:

```bash
npm run dev
```
